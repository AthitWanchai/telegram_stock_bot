# 📊 Telegram Stock Analysis Bot

บอทวิเคราะห์หุ้นอัจฉริยะบน Telegram ที่ช่วยให้คุณวิเคราะห์ตลาดหุ้นได้อย่างรวดเร็วและแม่นยำด้วยเครื่องมือทางเทคนิคที่ครบครัน

## 🌟 คุณสมบัติเด่น

### 📈 การวิเคราะห์เชิงเทคนิคครบครัน
- **RSI (Relative Strength Index)** - วัดแรงซื้อขายและจุดกลับตัว
- **MACD (Moving Average Convergence Divergence)** - สัญญาณการเปลี่ยนแปลงแนวโน้ม
- **Bollinger Bands** - ช่วงราคาและความผันผวน
- **EMA (Exponential Moving Average)** - แนวโน้มราคาระยะสั้นและยาว

### 🎯 การวิเคราะห์อัตโนมัติ
- **แนวรับ-แนวต้าน** - คำนวณจากราคาสูงสุดและต่ำสุด 20 วัน
- **สัญญาณซื้อขาย** - วิเคราะห์จาก RSI, MACD และ EMA
- **แนวโน้มตลาด** - ประเมินจากหลายปัจจัยร่วมกัน

### 📊 การแสดงผลที่เข้าใจง่าย
- สรุปผลการวิเคราะห์ด้วยอีโมจิและสี
- กราฟราคาพร้อมแนวรับ-แนวต้าน
- รองรับทั้งภาษาไทยและอังกฤษ

### 💼 รองรับสินทรัพย์หลากหลายประเภท
- **หุ้นไทย** - ใช้ `.BK` เช่น `PTT.BK`
- **หุ้นต่างประเทศ** - เช่น `AAPL`, `TSLA`, `GOOGL`
- **คริปโตเคอเรนซี่** - เช่น `BTC-USD`, `ETH-USD`
- **Forex** - เช่น `USDTHB=X`, `EURUSD=X`
- **ดัชนีตลาด** - เช่น `^SET.BK`, `^SPX`, `^DJI`
- **สินค้าโภคภัณฑ์** - เช่น `GC=F` (ทอง), `CL=F` (น้ำมัน)
- **กองทุน ETF** - เช่น `SPY`, `QQQ`, `VOO`

## 🚀 วิธีใช้งาน

### เริ่มต้นใช้งาน
1. พิมพ์ `/start` เพื่อเริ่มใช้งานบอท
2. ส่งชื่อหุ้นที่ต้องการวิเคราะห์ เช่น `PTT.BK` หรือ `AAPL`

### คำสั่งพิเศษ
- `/english` - สลับไปใช้ภาษาอังกฤษ
- `/thai` - สลับกลับมาใช้ภาษาไทย

### ตัวอย่างการใช้งาน
```
PTT.BK     - วิเคราะห์หุ้น PTT
AAPL       - วิเคราะห์หุ้น Apple
BTC-USD    - วิเคราะห์ Bitcoin
USDTHB=X   - วิเคราะห์ค่าเงินบาท
^SET.BK    - วิเคราะห์ดัชนี SET
```

## 📋 การติดตั้ง

### ความต้องการ
- Python 3.7+
- Telegram Bot Token
- แพ็คเกจ Python ตามไฟล์ `requirements.txt`

### ขั้นตอนการติดตั้ง
1. โคลน repository:
   ```bash
   git clone https://github.com/AthitWanchai/telegram_stock_bot.git
   cd telegram_stock_bot
   ```

2. ติดตั้ง dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. ตั้งค่า environment variables:
   ```bash
   cp .env.example .env
   # แก้ไขไฟล์ .env และใส่ Telegram Bot Token ของคุณ
   ```

4. เริ่มบอท:
   ```bash
   python telegram_stock_bot.py
   ```

## 🔧 การตั้งค่า

### สร้าง Telegram Bot Token
1. คุยกับ [@BotFather](https://t.me/botfather) บน Telegram
2. สร้างบอทใหม่ด้วยคำสั่ง `/newbot`
3. ตั้งชื่อและ username ให้กับบอท
4. บันทึก token ที่ได้รับ

### ตั้งค่า Environment Variables
สร้างไฟล์ `.env` และเพิ่ม:
```
TELEGRAM_BOT_TOKEN=your_bot_token_here
```
